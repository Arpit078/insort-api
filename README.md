# insort-api
web-scraping news api made using node js

# deployed on 
https://insort-api.herokuapp.com

# installation
  1. git clone https://github.com/Arpit078/insort-api.git
  2. cd insort-api
  3. npm install
  4. npm start{runs the nodemon server on port 5000}

# Usage
there are four routes :
   1. https://insort-api.herokuapp.com/Headlines
   2. https://insort-api.herokuapp.com/Business
   3. https://insort-api.herokuapp.com/Startup
   4. https://insort-api.herokuapp.com/Finance
